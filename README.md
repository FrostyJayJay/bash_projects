This is a collection of bash scripts and accompanying folders and files.

The purpose of these scripts are:

testd.sh - Simulate a downloads folder with multiple random file types that need to be sorted by their respective file extensions.
#Provided are the list of assorted folders and a bevy of different file types. Once the script is run all file extensions should be automatically sorted into their respective folder and a built in MacOS notification will play in top right corner.

redate.sh - Goes though a folder and adds todays current date to the files. Special care was give to ignore previous files that have already been dated so the script can be run daily if needed with out overwriting previous dates and a built in MacOS notification will play in top right corner.
#Provided are a list of files not containing dates for the script to act upon.
